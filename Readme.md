# 🌐 Personal Portfolio Website

This portfolio is a reflection of my work, ideas, and approach to building data-driven and geospatial solutions.

While the **vision, structure, and design direction** of this website were entirely conceptualized by me, I leveraged modern development tools — including agentic workflows and large language models — to **accelerate implementation and iteration**. These tools helped streamline development, but all key decisions around user experience, content, and presentation were intentionally designed.

The goal was to create a portfolio that is:
- Clean and intuitive
- Technically informative
- Visually engaging without being distracting
- Aligned with both academic and industry audience

---

## ⚙️ Tech & Approach

- Custom design adaptations and layout decisions
- Iterative development using AI-assisted tooling
- Focus on clarity, storytelling, and usability
